# Introducción a JavaScript

## Descripción
Este repositorio contiene ejercicios básicos realizados en JavaScript para practicar el uso de tipos de datos y el operador `typeof`.

## Objetivo
Aprender a identificar los diferentes tipos de datos que existen en JavaScript y practicar el uso de Git y GitHub.


### Práctica 1 - Tipos de datos
En esta práctica se utilizó el operador `typeof` para identificar diferentes tipos de datos en JavaScript como:
- Number
- String
- Boolean
- Undefined
- Null
- Object
- Array
- BigInt

### Práctica 2 - Evaluación de notas
En esta práctica se creó un programa que evalúa la nota de un estudiante utilizando estructuras condicionales `if`, `else if` y `else`.

El programa muestra un mensaje dependiendo de la calificación:
- Excelente
- Bien
- Suficiente
- No aprueba



## Tecnologías utilizadas
- JavaScript
- Git
- GitHub
