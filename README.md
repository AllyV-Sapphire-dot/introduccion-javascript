# Introducción a JavaScript

## Descripción
Este repositorio contiene ejercicios básicos realizados en JavaScript para practicar el uso de tipos de datos y el operador `typeof`.

## Objetivo
Aprender a identificar los diferentes tipos de datos que existen en JavaScript y practicar el uso de Git y GitHub.
